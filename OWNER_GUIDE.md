# Обновление контента

Материалы добавляет и заменяет только владелец сайта. У посетителей загрузка отключена.

## Музыка

1. Положите новый трек в эту папку проекта.
2. Либо замените `audio.mp3`, сохранив имя, либо укажите новое имя файла в `ownerAudio.src` в `script.js`.
3. В том же блоке измените `ownerAudio.name`, если нужно поменять подпись в плеере.

Самый надёжный формат для браузера - MP3. После замены файла обновите страницу.

## Файлы в менеджере

1. Папка `media` уже создана в корне проекта. Положите туда изображения или видео.
2. В `script.js` найдите объект `categoryFiles`.
3. Добавьте или обновите объект в нужной категории. Например:

```js
{ name: "Campaign still.png", type: "image", tone: "#f02b94", src: "./media/campaign-still.png" },
{ name: "Teaser.mp4", type: "video", tone: "#242631", src: "./media/teaser.mp4" },
```

Используйте `image` для PNG, JPG, WebP или GIF, и `video` для MP4 или MOV. Клик по карточке открывает слайд-шоу; у видео есть стандартные элементы воспроизведения.

Чтобы оставить плейсхолдер вместо настоящего файла, не указывайте `src`.
